rivs-inform.com
---------------------------------------------------------
(УСТАРЕЛО)
.../allData - post<Enterprise[], ProductElements[], lastDate[]> 
запрос данных для заполнения селекторов в интерфейсе
body {
"userLogin": 'login'
}
---------------------------------------------------------
(УСТАРЕЛО)
.../measure - post<measures>
запрос измерений
body {
"enterpriseName": 'someEnterprise',
"productName": 'someProduct',
"startDate": '',                    <----- можно отправить пустую строку
"endDate": ''			<----- можно отправить пустую строку
}
формат даты: 'yyyy-dd-MM'
---------------------------------------------------------
(УСТАРЕЛО)
.../products - post<ProductElements[]>
запрос продуктов для предприятия
body {
"enterpriseName": 'enterpriseName}'
}
---------------------------------------------------------
(УСТАРЕЛО)
.../updateDb - post<SynchroResult>
запрос синхронизации измерений из MariaDb(rt_meas) в SQL Server
body { 
"dateFrom": 'fromDate' <----- можно отправить пустую строку или null
}
