rivs-inform.com
---------------------------------------------------------
Build делался под IIS. Серверная часть в виде веб api
ASP.NET framework 4.8. БД MS Sql
---------------------------------------------------------
Логин пользователя и токен храниться в localstorage.
При открытии страницы измерений отправляется логин пользователя,
возвращается список предприятий, доступных пользователю,
список продуктов первого предприятия в списке,
и дата последнего измерения первого продукта в списке.
Затем отправляется полученная дата, первое предприятие и первый продукт предприятия,
ответом получаются измерения.
---------------------------------------------------------
.../allData - post<Enterprise[], ProductElements[], lastDate[]> 
запрос данных для заполнения селекторов в интерфейсе
body {
"userLogin": 'login'
}
---------------------------------------------------------
.../measure - post<measures>
запрос измерений
body {
"enterpriseName": 'someEnterprise',
"productName": 'someProduct',
"startDate": '',                    <----- можно отправить пустую строку
"endDate": ''			<----- можно отправить пустую строку
}
формат даты: 'yyyy-dd-MM'
---------------------------------------------------------
.../products - post<ProductElements[]>
запрос продуктов для предприятия
body {
"enterpriseName": 'enterpriseName}'
}
---------------------------------------------------------
.../updateDb - post<SynchroResult>
запрос синхронизации измерений из MariaDb(rt_meas) в SQL Server
body { 
"dateFrom": 'fromDate' <----- можно отправить пустую строку или null
}
